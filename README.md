Gravity_Claw is a sophisticated, secure personal AI agent framework with:

Purpose
A lean, secure, fully-understood personal AI agent system
TypeScript-based backend with a modern web-based control dashboard
Key Features
✨ AI Capabilities

Integration with Google Generative AI and OpenAI
Voice synthesis via Cartesia
Model Context Protocol (MCP) support for tool usage
📱 Communication

Telegram bot integration via Grammy
WebSocket support for real-time communication
🗄️ Data Management

PostgreSQL database integration (via Supabase)
SQLite for local data with better-sqlite3
Automatic task scheduling with node-cron
🎛️ Control Interface

Next.js-based Mission Control dashboard
React 19 frontend with Lucide icons
Tech Stack
Backend: TypeScript, Node.js
Frontend: Next.js 15, React 19
Databases: PostgreSQL (Supabase), SQLite
AI/LLM: OpenAI, Google Generative AI, Cartesia
Integrations: Telegram (Grammy), MCP SDK
DevOps: Docker, Docker Compose
